题解 [POJ 2456]Aggressive cows

方法：二分+贪心

这题我们可以用二分和贪心解决。

我们二分相邻距离的最小值，用这个值带进去贪心，第一个牛栏必选，之后如果某两个的牛栏大于现在的值就选，当任意时刻能放的牛大于要求的牛就 breakbreakbreak 且返回 truetruetrue，否则不满就返回 falsefalsefalse。

贪心的实现：

bool get (int mid) {
    //计数器cnt,j代表上次选的牛栏
	int cnt = 1;
	int j = 1;
    //最小的牛栏必选
	for (int i = 2;i <= n;i++) {
        //两个之间的区间满足条件
		if (a[i] - a[j] >= mid) {
			cnt++,j = i;
		}
		if (cnt >= m) {
			return true;
		}
	}
	return false;
}
