题解 [LightOJ 1240]Point Segment Distant(3D)

方法：三分

这题我们可以用三分解决。

我们三分直线 ABABAB，每次取 13\frac{1}{3}31​ 和 23\frac{2}{3}32​ 的两个位置计算距离并比较，不断往中间缩进。

直线 ABABAB 到 PPP 的距离的图像大约是：

我们把 13\frac{1}{3}31​ 位置记为 mid1mid1mid1 和 23\frac{2}{3}32​ 记为 mid2mid2mid2。

比较明显 mid1mid1mid1 到 ABABAB 的最短距离比 mid2mid2mid2 到 ABABAB 的最短距离更短，所以我们就可以将 mid2mid2mid2~BBB 的地方丢掉了。

重复该操作，直到 lll 和 rrr 足够接近即可。

在线段 ABABAB 上取点的做法：

s.x = a.x + (b.x - a.x) * mid;
s.y = a.y + (b.y - a.y) * mid;
s.z = a.z + (b.z - a.z) * mid;

两个三维坐标之间的距离：d=(x1−x2)2+(y1−y2)2+(z1−z2)2d=\sqrt{(x1-x2)^2+(y1-y2)^2+(z1-z2)^2}d=(x1−x2)2+(y1−y2)2+(z1−z2)2​。

具体见代码。

设 ABABAB 上某一点为 CCC，此代码三分 ACAB\frac{AC}{AB}ABAC​ 此比例。

Code:

#include<bits/stdc++.h>
using namespace std;
//考虑unsigned long long
typedef long long ll;
inline int read(){
	int z=0,x=1;char c=getchar();
	while(c>'9'||c<'0'){if(c=='-') x=-x; c=getchar();}
	while(c>='0'&&c<='9') z=z*10+c-'0',c=getchar();
	return z*x;
}
inline void write(ll x){
    if(x<0) putchar('-'),x=-x;
    if(x>9) write(x/10);
    putchar(x%10+'0');
}

const double eps = 1e-8;
struct node {
	double x,y,z;
}; 
node a,b,c;

//套用公式求该比例代表的点到 P 点的距离 
double get (double mid) {
	node s;
	s.x = a.x + (b.x - a.x) * mid;
	s.y = a.y + (b.y - a.y) * mid;
	s.z = a.z + (b.z - a.z) * mid;
	return sqrt((c.x - s.x)*(c.x - s.x)+(c.y - s.y)*(c.y - s.y)+(c.z - s.z)*(c.z - s.z));
}

int main () {
	int t = read();
	int cnt = 0;
	while(t--) {
		cnt++;
		a.x = read(),a.y = read(),a.z = read();
		b.x = read(),b.y = read(),b.z = read();
		c.x = read(),c.y = read(),c.z = read();
		//比例 
		double l = 0,r = 1;
		//三分比例 
		while(fabs(r - l) >= eps) {
			//获取1/3和2/3的比例位置 
			double mid1 = l + (r - l) / 3,mid2 = r - (r - l) / 3;
			//更新l和r 
			if (get(mid1) - get(mid2) < -eps) {
				r = mid2;
			} else {
				l = mid1;
			}
		}
		printf("Case %d: %.8lf\n",cnt,get(l));
	}
	return 0;
}
