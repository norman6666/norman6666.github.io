题解 [UVA - 1121]Subsequence

题解 [UVA - 1121]Subsequence

vjudge链接

404040 分做法：

用一个 sumsumsum 数组维护前缀和，暴力查找 [i,j][i,j][i,j] 间数的总和，与 sss 比较，如果大过就更新答案。

会被 TTT 掉。

满分方法：

方法一：尺取法

由于序列中的元素属于区间 [0,104][0, 10^4][0,104] ，所以前缀和肯定满足单调性。

定义一个 jjj 来枚举 111 到 nnn 的数。

 1  2  3  4  5  6  7  8  9
j1 j2 j3 j4 j5 j6 j7 j8 j9

举个例子，比如枚举到 j7j_7j7​ 的时候，我们可以找出一个满足条件的最短连续子序列 [i7,j7][i_7,j_7][i7​,j7​]，如继续向后枚举，不难发现，ik≥i7 (k>7)i_k ≥ i_7\ (k > 7)ik​≥i7​ (k>7)。

即 ik≥il (if jk>jl)i_k ≥ i_l\ (if\ j_k > j_l)ik​≥il​ (if jk​>jl​)。

我们就枚举 jjj，每次不用找过 iii 的值，只需要在之前的数据累加即可。

Code:

#include<bits/stdc++.h>
using namespace std;
//考虑unsigned long long
typedef long long ll;
inline ll read(){
	ll z=0,x=1;char c=getchar();
	while(c>'9'||c<'0'){if(c=='-') x=-x; c=getchar();}
	while(c>='0'&&c<='9') z=z*10+c-'0',c=getchar();
	return z*x;
}
const int N = 100010;
ll a[N],sum[N];

int main () {
	std::ios::sync_with_stdio(false);
	int n,s;
	while (~scanf("%d%d",&n,&s)) {
		for (int i = 1;i <= n;i++) {
			a[i] = read(); 
            //处理前缀和
			sum[i] = sum[i - 1] + a[i];
		} 
		int ans = 9999999;
		int i = 0;
        //枚举j,调整i
		for (int j = 1;j <= n;j++) {
			if (sum[j] - sum[i] < s) {
				continue;
			}
			while(true) {
                //找出满足条件的最大的i,为了找到符合题意的最短连续子序列的长度
				if (sum[j] - sum[i+1] >= s) {
					i++;
				} else {
					break;
				}
			}
            //更新
			ans = min(ans,j - i);
		}
		if (ans == 9999999) {
			cout << '0' << endl;
		} else {
			cout << ans << endl; 
		}
	}
	return 0;
}

方法二：二分

由于序列中的元素属于区间 [0,104][0, 10^4][0,104] ，前缀和肯定满足单调性，所以我们可以通过二分来做这道题。

从 111 到 nnn 枚举从当前位置 iii 的地方到地方 jjj （二分获取第一个、就是最小能满足条件的地方 jjj ），若满足条件，更新答案。

Code:

#include<bits/stdc++.h>
using namespace std;
//考虑unsigned long long
typedef long long ll;
inline ll read(){
	ll z=0,x=1;char c=getchar();
	while(c>'9'||c<'0'){if(c=='-') x=-x; c=getchar();}
	while(c>='0'&&c<='9') z=z*10+c-'0',c=getchar();
	return z*x;
}
const int N = 100010;
ll a[N],sum[N];

int main () {
	std::ios::sync_with_stdio(false);
	int n,s;
	while (~scanf("%d%d",&n,&s)) {
		for (int i = 1;i <= n;i++) {
			a[i] = read(); 
            //处理前缀和
			sum[i] = sum[i - 1] + a[i];
		} 
		int ans = 9999999;
		for (int i = 1;i <= n;i++) {
			int l = i,r = n,best = 0;
            //找到r所在位置
			while(l <= r) {
				int mid = (l + r) >> 1;
				if (sum[mid] - sum[i - 1] >= s) {
                    //更新
					best = mid;
					r = mid - 1;
				} else {
					l = mid + 1;
				}
			}
            //判断是为了排除找不到的情况，即best没有更新
			if (best != 0) {
				ans = min(ans,best - (i - 1));
			}
		}
		if (ans == 9999999) {
			cout << '0' << endl;
		} else {
			cout << ans << endl; 
		}
	}
	return 0;
}
